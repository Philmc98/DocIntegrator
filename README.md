# DocIntegrator — backend API

Минимальный демонстрационный проект на **.NET 8 / C#**.

## ?? Stack
- ASP.NET Core
- EF Core + PostgreSQL
- MediatR
- FluentValidation
- Swagger

## ? Getting started
1. dotnet restore
2. dotnet ef database update
3. dotnet run
4. Swagger: http://localhost:5000/swagger
